# Usage of AI

Here is list of things that AI agent was used for in this project:

- Building up README.md file
- Generating DTO models from sample API response
- Prototyping ImageInfoSheet
- Extracting piece of code to its own composable (function/file)
- Extracting hardcoded strings and dimensions to resources
- Generating kdoc for public/internal functions and classes
- Generating previews for composables
- Generating unit tests for repository
